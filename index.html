Biticker - Bitcoin Exchane Rates Widget

Download .zip Download .tar.gz View on GitHub

Biticker is a lightweight Bitcoin ticker, very easy to customize and embed in your website.

It is powered with data from bitcoinaverage.com

Just select the currency you want, choose a theme, and copy the generated code to your website.

That's it. Really.
